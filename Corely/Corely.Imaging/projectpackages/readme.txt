gsdll32.DLL is the 32 bit GhostScript library

It can be found by downloading and installing the 32 bit version of GhostScript and navigating to the bin folder of the install location

It is located here:
	C:\Program Files (x86)\gs\gs<version>\bin\gsdll32.DLL

Manually add it to the references of the C# project to use it
