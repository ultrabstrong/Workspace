ControllerInterop.DLL is the FlexiCapture interop library.

It can be found on any computer with FlexiCapture installed. 

It is located here:
	C:\Program Files\ABBYY FlexiCapture 12 Stations\ControllerInterop.dll

Manually add it to the references of the C# project to use it
