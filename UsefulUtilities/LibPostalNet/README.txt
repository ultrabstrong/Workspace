Full readme from libpostal in the LIBPOSTAL_README.md file. This can be viewed here : https://github.com/openvenues/libpostal

LibPostal needs ML-trained data files to accurately parse addresses. Data files can be downloaded by following the 'Installation (Windows)' instructions in the readme file.

This project was taken from the LibPostalNet solution on Github. The console application of the LibPostalNet solution shows some examples of how to use the LibPostal library in C#

This project must be built with x64 configuration because the postal.dll is compiled in x64. There are no official instructions for generating an x86 dll at this time.
